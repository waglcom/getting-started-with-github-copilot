# GitHub Copilot 시작하기

_1시간 이내에 GitHub Copilot의 핵심 사용 흐름을 연습합니다._

## 환영합니다

- **대상**: 코드 작업 흐름을 더 빠르게 만들고 싶은 모든 수준의 개발자
- **학습 내용**: Copilot으로 코드를 설명하고, 작성하고, 계획하고, 구현하는 여러 방법
- **만들 결과물**: Mergington High School의 방과 후 활동 웹사이트를 Copilot과 함께 개선합니다.
- **사전 준비**:
  - Skills 실습: [Introduction to GitHub](https://github.com/skills/introduction-to-github)
  - [VS Code](https://code.visualstudio.com/) 기본 사용 경험
  - 기본적인 코딩 개념
- **예상 시간**: 1시간 이내

이 실습에서는 다음을 진행합니다.

1. 미리 구성된 Codespace에서 브라우저 기반 VS Code를 실행합니다.
1. GitHub Copilot의 여러 상호작용 방식을 사용해 개발하고 계획합니다.
1. Copilot으로 pull request를 요약하고 리뷰합니다.

실습을 복사할 때는 다음 설정을 권장합니다.

- **Owner**에는 개인 계정 또는 저장소를 둘 조직을 선택합니다.
